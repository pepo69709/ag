# 🏹 Sniper AI V3.0: Command Center

この指令室は、最新のデータサイエンスとプレミアムなUIを融合させた、個人投資家のための意思決定支援システムです。

## 📂 ファイル構成と役割
- **`core.py`**: **【心臓部】** 指標計算、スコアリング、共通設定を管理。システムの「可用性」を支える根幹です。
- **`bot.py`**: **【目】** 市場をスキャンし、各銘柄の「星の数」を計算して `database.csv` に出力します。
- **`server.py`**: **【神経】** ブラウザ（UI）とデータを繋ぐバックエンドサーバー。ポートフォリオの記録を行います。
- **`index.html`**: **【指令画面】** 司令官が実際に操作するプレミアム・コンソール。
- **`auto_snipe_optimizer.py`**: **【フルオート】** 「特訓 -> 比較 -> 最強の適用 -> スキャン」を全自動で行う統合スクリプト。
- **`train.py`**: **【特訓】** 過去6ヶ月のデータから、今一番儲かる「重み」をAIに学習させます。
- **`compare_performances.py`**: **【審判】** 過去の知能と今の知能、どちらが「今」通用するかをガチンコ対決させます。
- **`restore_weights.py`**: **【レスキュー】** 万が一「前の重みのほうが良かった」と思った時に、瞬時に知能を復元します。

## 🏹 クイックスタート
1.  `python server.py` を起動
2.  ブラウザで `http://localhost:5000` を開く
3.  別のターミナルで `python bot.py` を実行して、最新の獲物をスキャン！

## 🛡️ 可用性と信頼性
- すべての計算ロジックは `core.py` に集約されており、一つの修正がすべてのツールに正しく反映されます。
- すべての学習（`train.py`）は、実行前に自動的にバックアップを取るため、知能の喪失リスクはありません。
